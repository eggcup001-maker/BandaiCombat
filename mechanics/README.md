【ゲーム仕様】
ゲームのカウントは99秒内で自機は10台までで終了。カウントは0から換算され99を超えるとゲームオーバー
5x4のLEDマトリクスに敵機と自機がそれぞれ１台ずつランダムに配置される。また地雷が2～3個配置される。
待機画面は上記で配置された状態で敵機(薄暗い)、自機(明るい)、現在のカウントが表示され、敵機0.5秒間隔くらいで点滅。

Fireボタンでスタート、カウントが1秒ずつ加算される。
上下左右ボタンで自機を移動し、敵の前でFireボタン＋方向で攻撃となる。Fireボタンを押さずに敵と同じマスに入った場合は自機を1台失う。
敵の上下左右又は斜めの、一つ隣に入ると約1秒間隔で敵の攻撃が来る(1回点滅で1攻撃)。
敵の攻撃が連続3回当たると自機を1台失う。自機を移動し隣ではなくなるとヒット回数がリセットされる。

ランダムに配置された地雷の上下左右一つ隣に入ると警告音が鳴る。
地雷を踏むと自機を1台失う。

攻撃が成功した場合は得点が1加算され、次の配置と待機モーションへ移行する。
敵の攻撃や接触、地雷で自機を1台失った場合は、その面の最初の配置にリセットされる(カウンタは増加し続ける)
ゲームオーバーで終了時、得点が表示され、終了。配置は次の面を配置する。スタートで得点を0点へリセットしカウントを0から開始でゲームスタート。

ゲーム進行時はカウントの様な音が鳴り続ける。待機中は無音。自機を失った場合は音が鳴る。
ゲームオーバー時は終了時のメロディーが鳴る。
